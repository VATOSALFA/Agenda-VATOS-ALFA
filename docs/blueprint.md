# **App Name**: Alfa Manager

## Core Features:

- Interactive Scheduler: Dashboard displaying barber schedules with drag and drop booking functionality
- Sales Management: Record and filter customer sales and payment methods.
- Appointment Reminders Dashboard: View appointment confirmation status with stats on sent, failed, and pending messages, so you can monitor overall system engagement.
- Customer Database: Searchable and filterable client database including quick customer creation.
- Inventory Management: Product inventory list showing Name, Category, Brand, Presentation, Price and Stock; allows for CSV upload.
- Reports Dashboard: Dashboard to show Occupancy Rate, Channel Revenue, and New Clients

## Style Guidelines:

- Primary color: Gold (#D4AF37) to reflect the VATOS ALFA brand and add a touch of elegance and sophistication.
- Background color: Dark gray (#333333) for a sleek, masculine dark theme.
- Accent color: Light gray (#B0B0B0) to provide subtle contrast and highlight key elements without overwhelming the dark background.
- Body and headline font: 'Inter' sans-serif with a modern, objective, neutral look for a masculine feel and excellent readability.
- Use sharp, geometric icons in gold and light gray to maintain a masculine, modern aesthetic.
- Fixed top navigation menu ensures key functions are always accessible. Columns on the Schedule page represent individual barbers.
- Subtle transitions and animations on hover and click events to enhance user experience without being distracting.
